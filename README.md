# AI RESUME ANALYZER

A Resume Analyzer web application built using Python and Streamlit.

This project compares a user's resume with a job description and provides:

- Resume Score
- Missing Skills
- Keywords to Add
- Suggestions for Improvement

## FEATURES

- Upload Resume PDF
- Paste Job Description
- Automatically identify missing skills
- Generate ATS-style resume score
- Suggest important keywords to improve the resume

## TECHNOLOGIES USED

- Python
- Streamlit
- PDFPlumber
- Basic NLP / Keyword Matching
